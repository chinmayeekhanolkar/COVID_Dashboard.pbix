# COVID-19 Dashboard (Power BI)

## Overview
This project is an interactive dashboard built using Power BI to analyze COVID-19 data.

## Features
- KPI cards (Confirmed, Deaths, Recovered, Active)
- Trend analysis using line chart
- Top 10 countries comparison
- Map visualization
- Interactive country slicer

## Insights
- Countries with highest cases dominate totals
- Recovery trend increases over time
- Death rate is lower compared to confirmed cases
- Active cases vary across countries

## Tools Used
- Power BI
